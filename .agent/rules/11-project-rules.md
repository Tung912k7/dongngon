# Project Specific Rules

Rules stringently enforced for this specific project.

## 1. No AI-Generated Images

- **Strict Prohibition:** The use of AI-generated images (AI-generated art, photos, etc.) is strictly prohibited on the platform and within the codebase.
- **Originality:** Any assets used must be originally created, properly licensed, or stock images without the use of AI generation.
- **Agent Enforcement:** The agent must never generate, use, or propose adding AI-generated images to the project.
